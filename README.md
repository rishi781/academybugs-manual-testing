## Overview
This is a **Manual Testing Project** for the **AcademyBugs** demo e-commerce website ([https://academybugs.com/find-bugs/](https://academybugs.com/find-bugs/)).  
It demonstrates end-to-end manual testing including **test planning, test cases.**.

## Project Scope
- Homepage: Product display, sorting, filtering, pagination  
- Product Details Page: Description, social media buttons, feedback form  
- Cart Management: Add/remove items, update quantities, dynamic total  
- Checkout & Payment: Billing/shipping forms, coupons, proceed to checkout  
- User Account: Signup, login, dashboard, address management  

## Types of Testing Used
- **Functional Testing:** Verify each feature works as expected  
- **Usability Testing:** Ensure the website is user-friendly  
- **Exploratory Testing:** Identify unexpected bugs  
- **Regression Testing:** Re-test after fixes to ensure stability  
- **Integration Testing:** Ensure modules like cart and checkout work together  
- **System Testing:** Validate the entire application end-to-end  

## Deliverables
- Test Plan  
- Test Cases   
- Screenshots
